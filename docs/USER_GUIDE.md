## 登录

- 地址：开发环境 http://localhost:5173，Docker http://localhost:8080
- 首次部署后执行 `npm run seed`，使用 `.env` 中配置的 `SEED_ADMIN_PASSWORD` 登录管理员账号 `admin`

## 核心业务流程

### 采购流程

1. **采购申请** → 创建并提交审批
2. **转采购单** → 审批通过后选择供应商生成采购订单
3. **审批采购订单** → 草稿订单审批
4. **收货入库** → 选择仓库完成收货
5. **应付管理** → 自动生成应付单，登记付款

### 销售流程

1. **公海线索** → 筛选沈阳 POI 商户 → **领取** 至我的线索（上限 50 条）
2. **联系上报** → 填写跟进方式、联系结果、摘要（可选录音）→ 提交待审
3. **上报审核**（销售总监/管理员）→ 通过或驳回 → 查看联系统计
4. **销售报价** → 创建报价单并审批
5. **转销售单** → 生成销售订单
6. **审批并发货** → 审批后选择仓库发货（扣减库存）
7. **应收管理** → 自动生成应收单，登记收款
8. **售后服务** → 创建售后工单并跟踪处理

### 库存管理

| 功能 | 说明 |
|------|------|
| 库存台账 | 查看各仓库库存 |
| 出入库 | 手动调整库存 |
| 库存调拨 | 仓库间转移库存 |
| 库存盘点 | 账面 vs 实盘，完成自动调整 |
| 库存预警 | 低于安全库存的产品列表 |
| 产品管理 | 设置安全库存阈值 |

### 生产流程

1. **BOM 管理** → 维护物料清单
2. **生产计划** → 制定生产计划
3. **工单管理** → 下达、开工、完工入库
4. **质量管理** → 质检记录

### 财务模块

- **总账**：科目维护、凭证录入与过账
- **应收/应付**：由业务单据自动生成
- **固定资产**：资产登记、计提折旧、处置
- **财务报表**：科目余额表、应收应付汇总

### 人力资源

员工、部门、岗位、考勤、薪资、绩效管理。

### 系统集成

系统管理 → 系统集成，提供 REST 导出接口供外部系统对接（需 `integration:sync` 权限）。

### 审计日志

系统管理 → 审计日志，可查询：

| 类别 | 说明 |
|------|------|
| auth | 登录成功/失败 |
| operation | 数据增删改（POST/PATCH/DELETE） |
| security | 权限拒绝 |

支持按用户、时间、操作类型筛选。

### 多租户与预算

- **租户管理**（系统管理 → 租户管理）：创建租户，登录时使用租户编码（默认 `default`）
- **预算管理**（财务管理 → 预算管理）：按科目/部门设置预算，采购订单审批时自动扣减预算额度

### 批次追溯

库存管理 → **批次追溯**：输入批次号或序列号，查看采购收货、调拨、销售发货等流转记录。

### 公海线索导入

管理员 → **系统管理 → 线索导入**（CSV / JSON），或运维脚本 / API 批量写入。  
完整说明见 **[详细使用说明 7.10 节](USER_MANUAL.md#710-公海数据导入管理员)**。

### 文件中心

系统管理 → **文件中心**：上传附件至 MinIO 对象存储（需配置 `MINIO_*` 环境变量），支持预签名下载链接。

### 实时通知

Header 铃铛图标可查看站内通知；后端通过 WebSocket 推送新消息（无需手动刷新页面）。

### BI 大屏

侧栏 **BI 大屏** 或访问 `/bi-screen`：全屏运营数据可视化，每 30 秒自动刷新，支持全屏展示。

### 语言切换

登录页或 Header 右上角可切换 **中文 / English**。导航菜单与面包屑会随语言切换；部分业务页面仍为中文。

## 上线培训要点

1. **管理员**：用户/角色/权限配置、系统参数、审计日志查看
2. **采购员**：采购申请 → 采购订单 → 收货流程
3. **销售员**：公海领取 → 联系上报；报价 → 销售订单 → 发货 → 售后工单
4. **销售主管**：上报审核、联系统计、团队线索管理
5. **仓管员**：库存台账（含批次号）、出入库、调拨、盘点
6. **财务**：总账凭证、应收应付、固定资产、财务报表

建议按角色分批培训，每批 1–2 小时，配合上述业务流程实操。

## 权限说明

- 侧边栏仅显示有权限的菜单
- 直接访问无权限 URL 将显示 403 页面
- 角色权限在 **系统管理 → 角色管理** 中配置

## 常见问题

**Q: 发货提示库存不足？**  
A: 检查 **库存台账** 中对应仓库是否有足够库存，或通过采购收货/生产完工补充库存。

**Q: 库存预警无数据？**  
A: 在 **产品管理** 中为产品设置 **安全库存** 大于 0。

**Q: 报表数据不是最新的？**  
A: 报表有 1–2 分钟缓存，稍等刷新即可；生产环境配置 Redis 后多实例共享缓存。

**Q: 联系上报后主管看不到？**  
A: 主管需 `lead:review` 权限并访问 **销售管理 → 上报审核**；销售专员无审核权限（403 属正常）。

**Q: 录音无法上传？**  
A: 启动 MinIO 并配置 `backend/.env` 中 `MINIO_*` 变量；文件须为音频且 ≤ 10MB。
